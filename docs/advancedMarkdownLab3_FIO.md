#### ALERT-БЛОК
> [!TIP]
> Совет по использованию: Для экранирования специальных символов в Markdown используйте обратный слэш. Например, чтобы написать *курсив* баз форматирования, напишите \\*курсив\\*.

#### ESCAPE-СИМВОЛЫ
• Заголовок без форматирование: \# Это не заголовок
• Звездочка как текст: \*это просто звёздочка\*
